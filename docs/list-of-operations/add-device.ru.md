# Add device

Привязать устройство к аккаунту.

Данная операция перемещает 0.1 XAT от аккаунта-источника в системный пул [`txFeePool`][1]

Threshold: medium


## Параметры

- `device public key` - публичный ключ устройства, которое необходимо привязать
